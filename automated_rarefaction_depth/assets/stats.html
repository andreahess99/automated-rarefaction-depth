Automated Rarefaction Depth Summary

This visualization shows the rarefaction analysis for the beta diversity metric {{ beta_metric }}. The Rarefaction Curve plot on the left shows the average differences between the distance matrices at the different read depths. Each distance matrix was compared to the previous one. The curve shows how the average difference changes with increasing read depth.
The optimal rarefaction depth is depicted as the vertical red line. It is the knee point that was calculated using the {{ algorithm }} algorithm.

Knee point: {{ knee_point }}

The bar plot on the right shows how many samples were included at each evaluated read depth.
